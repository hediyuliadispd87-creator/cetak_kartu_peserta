Kartu Peserta Ujian

Upload file Excel atau CSV untuk mulai membuat kartu peserta.

1
Klik "Upload Spreadsheet" di toolbar atas, pilih file .xlsx atau .csv
2
Pilih sheet lalu klik "Load Sheet" — atau "Semua Kelas" untuk load sekaligus
3
Kartu tersusun otomatis 8 per halaman A4 (2 kolom × 4 baris)
4
Klik "Edit Kop & TTD" → unggah cap & tanda tangan, atur posisi dengan slider
5
Klik "Cetak" untuk cetak langsung, atau "Download PDF" untuk simpan file PDF
